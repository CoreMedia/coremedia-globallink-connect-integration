Prerequisites

Ensure you have your GCC parameters at hand:

  • GCC REST Base URL
  • API Key
  • File Type
  • Optional: Client Secret Key (see below)

For more details on available options and how to configure them in CoreMedia Studio visit Configuring Connection Settings.
